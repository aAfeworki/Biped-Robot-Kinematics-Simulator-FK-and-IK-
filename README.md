# Biped-Robot-Kinematics-Simulator-FK-and-IK-
Interactive Python-based simulators for a biped robot, including forward kinematics, inverse kinematics with fixed base, and inverse kinematics with floating base.

This repository provides interactive simulators for a bipedal robotic system, implemented in Python using numpy, matplotlib, and tkinter.
It demonstrates both forward and inverse kinematics, making it a valuable tool for exploring the motion, posture, and control of legged robots.

🚀 Features

Forward Kinematics 

Control the biped robot’s joint angles interactively using sliders to visualize the resulting pose of the robot.

Inverse Kinematics Simulator (Fixed Base)

Solve joint angles for a desired foot position while keeping the base fixed.

Inverse Kinematics Simulator (Floating Base)

Solve joint angles for a desired floating base configuration, enabling more realistic posture exploration.
